# Telegram Bot for TwGame

Телеграм-бот для игры TwGame. Бот служит точкой входа в игровой веб-интерфейс.

## Установка и запуск

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите бота:
```bash
python main.py
```

## Функционал

- Команда `/start` - приветственное сообщение с кнопкой "Играть"
- Кнопка "Играть" - пока показывает заглушку, позже будет открывать веб-интерфейс игры

## Структура проекта

- `main.py` - основной скрипт бота
- `requirements.txt` - зависимости Python
- `README.md` - это файл
